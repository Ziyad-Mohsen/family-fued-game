How to Play

Game Overview

  1. Teams: Divide players into two teams. Each team takes turns guessing answers to survey questions.
  2. Objective: Guess the most popular answers to survey questions to earn points.
  3. Rounds: The game consists of multiple rounds. Each round has one survey question with several hidden answers ranked by popularity.

Setup

  1. Host Page: Open host.html in a separate window or device. It displays all survey questions and answers for the host only.
  2. Game Board: Use index.html on the main display for players. It shows the answer board and strike indicators.
  3. Recommended Setup: Use two screens/devices—one for the host and one for the players.

How to Play

  1. The host reads a survey question from host.html aloud.
  2. Teams alternate guessing answers. If a guessed answer is correct, the host reveals it on index.html.
  3. If a team guesses incorrectly, they receive a strike (up to 3).
  4. The round ends when all answers are revealed or one team accumulates 3 strikes.
  5. Points are awarded based on the value of revealed answers. The team with the most points at the end wins!

Host Controls (Keyboard Shortcuts)

  • Reveal Answers: Press keys 1–5 to show the corresponding answer slot.
  • Show Strikes: Press Q, W, or E to display 1, 2, or 3 strikes.
  • Next Round: Press N to go to the next question and play the round intro sound.
  • Previous Round: Press P to return to the previous question.
  • Play Intro Sound: Press I.

Extra Features

  • Sound Effects: Sounds play for correct answers, wrong answers, and round changes.
  • Language Support: Questions and answers support English and Arabic.
  • Answer Display: When revealed, answers appear visually on the main game board for all to see.

Tip: For best results, use two devices: one for the host (to control questions and answers) and one for displaying the game board to players.
